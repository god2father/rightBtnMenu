# Win10 Right Click Menu Manager

一个尽可能简单直接的 Windows 10 右键菜单管理工具。

当前实现特点：

- 不依赖第三方库
- 直接用 PowerShell + WinForms 做本地 GUI
- 默认新增写入 `HKCU\Software\Classes`，通常不需要管理员权限
- 列表读取的是系统当前实际生效的右键菜单（`HKCR` 合并视图）
- 支持查看、添加、删除、启用、禁用常见右键菜单项
- 保存、删除、启用/禁用后会自动刷新资源管理器右键菜单缓存
- 选择已注册程序时会尽量解析为真实可执行文件路径，VSCode 会自动生成更稳的启动命令
- 支持在保存前先点击 `测试命令`

支持的右键范围：

- 文件
- 文件夹
- 文件夹空白处
- 桌面空白处

## 启动

优先双击 `打开右键菜单管理.cmd`

如果你希望以后不用进项目目录，双击 `安装快捷方式.cmd`，它会自动在桌面和开始菜单创建 `Right Click Menu Manager` 快捷方式。

如果要看启动过程或排查报错，再双击 `调试启动.cmd`

兼容旧入口：

- `start.bat` 会转发到 `打开右键菜单管理.cmd`

或者命令行执行：

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -STA -File .\rightBtnMenu.ps1
```

## 使用方式

1. 选择右键菜单作用范围
2. 点击 `新增` 进入新增模式
3. 可直接点击 `选择程序`，从已注册程序列表中挑一个
4. 自动填入菜单名称、命令、图标后，可先点 `测试命令`
5. 确认没问题后点击 `新增保存`
6. 如果是修改列表里当前选中的项目，直接编辑后点 `更新保存`
7. `更新保存` 仅直接更新当前用户创建的菜单项；系统菜单建议用 `新增保存` 新建自己的版本
8. 也可以手动填写菜单名称 `Menu Text`、命令 `Command`、图标 `Icon`
9. 未选中菜单项时，`更新保存`、`删除`、`禁用/启用` 会自动置灰；系统项会显示为只读

删除时：

1. 在列表里选中现有项目
2. 点击 `Delete`

禁用或重新启用时：

1. 在列表里选中现有项目
2. 点击 `Disable` 或 `Enable`

## 命令示例

对文件打开记事本：

```powershell
notepad.exe "%1"
```

对文件夹打开终端：

```powershell
powershell.exe -NoExit -Command Set-Location "%V"
```

用 VSCode 打开当前文件夹：

```powershell
Code.exe --reuse-window "%V"
```

说明：

- 文件/文件夹类菜单通常用 `%1`
- 文件夹空白处/桌面空白处通常用 `%V`
- 这是最小可用版本，暂时没有做导入导出、排序、系统级 `HKLM` 菜单管理等扩展能力
